# Task2_Web_Development

Files:
- index.html <br />
HTML instructions for initial page (My solution)
- 2Dnetwork.html <br />
HTML sheet for 2D graph page
- 2Dnetwork.js <br />
JavaScript script for 2D graph visualisation in 2D graph page
- 3Dnetwork.html <br />
HTML sheet for 2D graph page
- 3Dnetwork.js <br />
JavaScript script for 2D graph visualisation in 2D graph page
